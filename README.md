# Vusui Icon 图标库

集成 500+ 个常用的图标。



【[下载图标库](https://github.com/vusui/vusui-icon)】|【[在线预览](https://cdn.vusui.com/icon/5.0)】



## 线上调用方法

```html
# 在 html 文件 head 引入
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <link rel="stylesheet" href="https://cdn.vusui.com/icon/5.0/index.css" />
  </head>
  <body></body>
</html>
```



## 下载资源包到项目

```
# 引入方法
import './vusui-icon/index.css';
```
